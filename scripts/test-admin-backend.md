# 管理員後台功能測試

## 測試步驟

### 1. 登入測試
1. 開啟 http://localhost:3001/login
2. 在用戶登入標籤中輸入：
   - 用戶名：`guard`
   - 密碼：`guard`
3. 點擊登入
4. 應該跳轉到 `/admin/dashboard`

### 2. 導航測試（行動版）
1. 將瀏覽器調整為行動裝置尺寸（寬度 < 768px）
2. 檢查底部導航列是否顯示 5 個按鈕：
   - 總覽
   - 即時消息  
   - 用戶數據
   - 店家數據
   - 訂單
3. 點擊每個按鈕確認頁面正常載入

### 3. 儀表板測試（行動版）
1. 在總覽頁面檢查統計卡片是否以 2x2 網格顯示：
   ```
   [用戶總數] [店家總數]
   [進行中訂單] [訂單完成率]
   ```

### 4. 各頁面功能測試
- **總覽**：檢查統計數據顯示
- **即時消息**：檢查新聞列表
- **用戶數據**：檢查用戶列表
- **店家數據**：檢查店家列表  
- **訂單**：檢查訂單列表

### 5. 桌面版測試
1. 將瀏覽器調整為桌面尺寸（寬度 >= 768px）
2. 檢查頂部導航列是否正常顯示
3. 檢查儀表板是否顯示為 4 欄佈局

## 預期結果
- ✅ 所有頁面正常載入，無錯誤
- ✅ 響應式設計正常工作
- ✅ 導航功能正常
- ✅ 數據正常顯示
